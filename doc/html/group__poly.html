template<typename vec_type >
cx_vec arma_ext::roots ( const vec_type &  c)

Polynomial roots.

The polynomial \(s^3 - 6s^2 - 72s - 27\) is given then, the roots of this polynomial are returned in a column vector.

vec p = "1 -6 -72 -27";
cx_vec r = roots(p);
real(r).print("r = "); // the polynomial p has real roots.

The output is \( r = \begin{matrix} 12.1229 \\ -5.7345 \\ -0.3884 \end{matrix} \)

Parameters
cThe polynomial coefficient vector.
Returns
A column vector whose elements are the roots of the polynomial c.
