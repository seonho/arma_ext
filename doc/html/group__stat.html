2-D correlation coefficient

Parameters
A
B
Returns
the correlation coefficient between A and B, where A and B are matrices or vectors of the same size. corr2 computes the correlation coefficient using

\[ r = \frac{ \sum_{m}\sum_{n}(A_{mn}-\bar{A}) (B_{mn}-\bar{B}) }{\sqrt{ \left( \sum_{m}\sum_{n}(A_{mn}-\bar{A})^2 \right) \left( \sum_{m}\sum_{n}(B_{mn}-\bar{B})^2 \right)}} \]

where \( \bar{A}\)=mean2(A), and \(\bar{B}\)=mean2(B).
