Parameters
XA matrix.
nThe count of leading dimensions to be shifted.
Note
arma::Mat only support 2D array. If X is a row vector, then shiftdim returns a column vector.
If X is a matrix, then shiftdim returns a matrix that leading non-singleton dimensions.
