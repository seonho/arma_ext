enum arma_ext::convolution_type

Convolution types

Enumerator
full 
same 

The full two-dimensional convolution (default).

valid 

The central part of the convolution of the same size as A.

Only those parts of the convolution that are computed without the zero-padded edges.
Using this option, size (C) = max([ma - max(0, mb - 1), na - max(0, nb - 1)], 0).
