Difference and approximate derivative.

Template Parameters

Parameters
XA vector or a matrix.
nThe order of differentiation.
dim(only for matrix) The dimension to be differentiated along.
Returns
If X is a vector, then diff(X) returns a vector, one element shorter than X, of differences between adjacement elements:

\[ \begin{bmatrix} X(2) - X(1) & X(3) - X(2) & \cdots & X(n) - X(n - 1) \end{bmatrix} \]

If X is a matrix, then diff(X) returns a matrix of row differences:

\[ \begin{bmatrix} X(2:m, :) - X(1:m - 1, :) \end{bmatrix} \]

In general, diff(X) returns the difference calculated along the first non-singleton (size(X, dim) > 1) dimension of X.

diff(X, n) applies diff recursively n times, resulting in the nth difference. Thus diff(X, 2) is the same as diff(diff(X)).

diff(X, n, dim) is the nth difference function calculated along the dimension specified by scalar dim. If order n equals or exceeds the length of dimension dim, diff returns an empty array.
